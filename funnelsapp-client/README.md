This is the code base for my React app.

## 1: API Base URL

Add https://us-central1-funnel-app-bdd85.cloudfunctions.net/api as the 'proxy' value in package.json

## 2: Install packages

run `npm install`

## 3: Run project

run `npm start`

## 4: Open it

go to [http://localhost:3000](http://localhost:3000)
